PRE-INSTALLATION
----------------
- create NODEJITSU account
- install NODEJITSU CLI

INSTALLATION
------------
- unzip JAP_REMOTE_WS_NODE_NODEJITSU-X.X.X.zip
- open JAP_REMOTE_WS_NODE_NODEJITSU-X.X.X
- configure JAP_REMOTE_WS.json
- open CONSOLE
	- cd JAP_REMOTE_WS_NODE_NODEJITSU-X.X.X
	- jitsu login
	- jitsu deploy
